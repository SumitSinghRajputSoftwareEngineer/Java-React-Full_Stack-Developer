
## Definition

`@Null` is a Bean Validation annotation used to validate that a field value is:

```text
null
```

It tells the validation framework:

> The field must be null.

Package:

```java
import jakarta.validation.constraints.Null;
```

Prior to Spring Boot 3:

```java
import javax.validation.constraints.Null;
```

---

# Why Do We Need @Null?

Sometimes clients should **not provide a value** for certain fields.

The system itself will generate or populate those values.

Examples:

```text
id
createdDate
createdBy
updatedDate
version
audit fields
system generated fields
```

---

Example:

Client sends:

```json
{
    "id": 100
}
```

But:

```text
ID should be generated by database.
```

---

This request should fail.

---

Solution:

```java
@Null
```

---

# Common Mistakes

## Mistake 1

Using @Null Instead Of @NotNull

Wrong:

```java
@Null
private Long id;
```

when id is mandatory.

---

Use:

```java
@NotNull
```

---

# Mistake 2

Expecting Empty String To Pass

Wrong assumption:

```java
""
```

is null.

---

Actually:

```java
"" != null
```

---

Validation fails.

---

# Mistake 3

Using @Null On Entity

Usually validation belongs in:

```text
DTO
```

not:

```text
JPA Entity
```

---

Keep validation close to API layer.

# Common Interview Questions

## What is @Null?

## Does @Null allow null?

## Does @Null allow empty string?

No.

---

## Does @Null allow 0?

`@Null` requires null.

# Key Points To Remember

- `@Null` validates that a value must be null.
- Commonly used for system-generated fields.
- Frequently used for IDs, audit fields, and version fields.
- Opposite of `@NotNull`.
- Missing JSON fields generally become null and pass validation.
- Empty string (`""`) is not null.
- Can be combined with Validation Groups.
- Internally uses `NullValidator`.
- Very common in Create vs Update API validation.
- Frequently asked Spring Boot and Hibernate Validator interview annotation.

---

# Quick Comparison

| Annotation | Valid Value |
|------------|------------|
| `@Null` | `null` |
| `@NotNull` | Any non-null value |
| `@NotBlank` | Non-empty text |
| `@NotEmpty` | Non-empty collection/string |
